1.5

Copyright laws differ slightly all over the world, so there are steps you should do to keep your work protected and be sure you are honoring the copyrights of others. The code of the page should have a notice somewhere inside declaring your copyright over the material, in visible text. Or a link to your notice visible somewhere on the page.. Your images need the copyright symbol pasted in a corner. And any file downloaded or sent from your website should have it in invisible text. When creating a webpage for a third party, make sure all copyrights are transferred ownership correctly, as well being sure to register your website’s copyright online. If you are interested, tools can also be used to hunt for infringements of your site’s copyright, where you can then decide to move on with takedown notices to enforce your hold of the intellectual property.

1.6

A multitude of disabilities and special cases should be considered when making a website. There are those that need to use the page with limited mobility in their hands or must use the interface in a more abstract manner. So making sure the content is easily navigated and links are easily accessed can make it easy for people to get through your page with as little input as possible. Sight is a major thing to keep in mind as well, as those with certain colour blindnesses will need the page formatted in such a way so that colours that fade together in their eyes don’t make the page indecipherable. Or have a colour-blind mode that is easily switched to. Also keeping text as plain text instead of imbedded into an image can keep the page readable by a screen-reader for those that are completely blind.

Many file types exist, but here are some of the most commonly used:
AVI - Audio/Video Interleaved: Standard windows video format. Cannot be streamed.
CSS - Cascading StyleSheet: Tools that webmasters use to take care of how the website loos
DOC - Microsoft Word Document: Document file used by Microsoft Office’s Word program.
TXT - Text File: Most basic file, stores simple text
EXE - Executable Application: Executable program, mostly used to install and run programs
GIF - Graphics Interchange Format: Most common image on the internet, used for simple images and can also be used for moving images.
JPEG - Joint Photographic Experts Group file: Another image file type that is commonly used.
PNG - Portable Network Graphics: File format used in place of GIF, usually smaller and have more advanced features.
MPEG -Motion Picture Experts Group file: Standard video streaming file
MP3 - MPEG Layer 3 Sounds file: Highly compressed audio file, allowing for easily downloaded audio with great quality.
PDF -Portable Document Format: Adobe Acrobat file format used to send documents that remain exactly the same between platforms and businesses.
ZIP - Zipped File: A compressed file used to more easily send folders and groups of files more quickly.
